# eICU-CRD-2.0 Layer 1 Manifest

This database skeleton follows the shared Layer 1 contract:

- `raw_original`
- `raw_unpacked`
- `source_supplied_derived`
- `local_converted_parquet`
- `docs_manifest`

The current local intake stages the eICU Collaborative Research Database 2.0 delivery under `raw_original`.

Restricted raw CSV/TXT files must remain outside this public repository. Public files here are only the data-free skeleton and documentation.

Historical owner workspaces for eICU are reference-only and are not part of this public or production Layer 1 skeleton.

Layer 2 opening is intentionally deferred until the current multi-database intake wave is ready for a coordinated opening pass.
